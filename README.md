# Mobile-sales-data-dashboard

I built this dashboard to understand mobile sales performance across brands, cities, and time - in a way that’s quick and actually useful

Problem it solves :

It’s usually hard to track which brands, cities, or time periods are actually driving sales and how performance is changing over time. This dashboard makes it easy to see sales trends, brand performance, and year-over-year comparison - all in one place.

What I focused on :

1. Total sales, transactions, and quantity
2. Brand and model performance
3. City-wise sales distribution
4. Payment method split
5. Customer ratings
6. MTD (month-to-date) and SPLY (last year comparison)

Key Insight :

1. Apple leads, but Samsung is almost equal — tight competition
2. Sales peak around mid-year (July) and drop towards early months
3. Payment methods are evenly split — no single dominant mode
4. Most customers give “Good” ratings, very few bad experiences

One clear trend :

Sales consistently perform better in mid-year months, and current year performance is slightly lower than last year - something to watch closely.
